Increment Button in Hyperscript

By Mico Monks

Here's an example of something we had to do in JavaScript, that is much easier in Hyperscript! The code reads as follows: 

        
            <button _="on click increment :x
                if :x > 10
                    set :x to 0
                    put :x into the next <output/>
                else
                    put :x into the next <output/>
                end">
            Click Me
            </button>

This allows us to create a button to increment a number up until 10 directly in the html rather than in a separate JavaScript file.

--

We can even add that second button that increments by 2. I just did this by putting two increment statements into the button.
