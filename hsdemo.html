Welcome to Hyperscript!

By Mico Monks

In Hyperscript, we can add script directly to the element itself. For instance, we can change the color of this button on click, without the use of an event handler. As long as we have the id ".red" defined in css, we can toggle it on and off.

The code for this button looks like this: 

        
            <button script="on click toggle .red on me">
                Click Me
            </button>

But we can do more than just buttons. Since Hyperscript works with elements, we can actually change the color of the entire div! Try clicking on the code below. 

        
            <div _="on pointerdown
                            repeat until event pointerup
                                set rand to Math.random() * 360
                                transition
                                    *background-color
                                    to `hsl($rand 100% 90%)`
                                    over 50ms
                            end">

All we're doing here is adding a few lines of Hyperscript to the div that all this text is in, and it changes the background color of everything inside the div! Also note how we can replace "script" with "_".
